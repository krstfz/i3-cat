# i3wm
i3wm catppuccin setup (+ everforest // dracula // gruvbox // nord)
* wm - i3
* bar - i3blocks
* spotify - spicetify 
* launcher - rofi
* fonts - JetBrainsMono Nerd Font, Ubuntu Nerd Font, Fira Code Mono
